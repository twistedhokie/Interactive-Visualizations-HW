Latitude vs. the Weather

The purpose of this project was to analyze the effect of latitude on certain weather conditions. We utilized the OpenWeatherMap API to gather data from over 500 cities worldwide.
This collection of pages highlights the findings and provides visualizations for each plotted relationship. Overall, The only significant relationship that was evident was latitude vs. maximum temperature. This is somewhat to be expected, as it generally gets warmer as you move towards the equator.
